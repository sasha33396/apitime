Домены
В Timeweb Cloud доступна регистрация и продление доменов более чем в 350 зонах.

В зависимости от выбранной зоны, домен может быть зарегистрирован через регистратора Таймвэб.Домены или через одного из наших партнеров.

Подробнее о доменах – в нашей базе знаний.

Получение списка всех доменов
Чтобы получить список всех доменов на вашем аккаунте, отправьте GET-запрос на /api/v1/domains.

Тело ответа будет представлять собой объект JSON с ключом domains.

Авторизация:
Bearer
query Parameters
limit	
integer
По-умолчанию: 100
Обозначает количество записей, которое необходимо вернуть.

offset	
integer
По-умолчанию: 0
Указывает на смещение относительно начала списка.

idn_name	
string
Пример: idn_name=xn--e1afmkfd.xn--p1ai
Интернационализированное доменное имя.

linked_ip	
string
Пример: linked_ip=192.168.1.1
Привязанный к домену IP-адрес.

order	
string
Enum: "asc" "dsc"
Пример: order=asc
Порядок доменов.

sort	
string
Значение: "idn_name"
Пример: sort=idn_name
Сортировка доменов.

Ответы
200 Ответ будет представлять собой объект JSON c ключом domains.
400 Некорректный запрос
401 Не авторизован
429 Слишком много запросов
500 Внутренняя ошибка сервера
АвторизацияВыполнить запрос
get
/api/v1/domains
Примеры запроса
cURLJavaScriptNodePythonPHPGoJava

Копировать
package main

import (
	"fmt"
	"io"
	"log"
	"net/http"
	"os"
)

func main() {
	client := &http.Client{}
	req, err := http.NewRequest("GET", "https://api.timeweb.cloud/api/v1/domains", nil)
	if err != nil {
		log.Fatal(err)
	}
	req.Header.Set("Content-Type", "application/json")
	req.Header.Set("Authorization", "Bearer " + os.Getenv("TIMEWEB_CLOUD_TOKEN"))
	resp, err := client.Do(req)
	if err != nil {
		log.Fatal(err)
	}
	defer resp.Body.Close()
	bodyText, err := io.ReadAll(resp.Body)
	if err != nil {
		log.Fatal(err)
	}
	fmt.Printf("%s\n", bodyText)
}
